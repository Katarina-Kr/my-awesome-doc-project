# Создать группу 

> **Warning:** Для S25 FE Android 16.

Чтобы создать группу:

1. Нажмите **Три точки** в правом верхнем углу экрана.

2. Нажмите  **Создать группу**.

3. Добавьте минимум одного участника группы.

4. Введите название группы.

5. При желании установите фото или эмодзи для группы.

6. Нажмите на **Галочку**.

7. При желании включите автоудаление сообщений и выберите время, через которое сообщения будут удаляться.
